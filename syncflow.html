프로젝트 개요

SyncFlow가 무엇이고, 무엇을 해결하는가

Jira + Confluence + Slack + Zoom을
하나의 플랫폼으로 통합

기업 팀이 프로젝트를 진행할 때 필요한 문서 편집, 팀 채팅, 음성 회의, 작업 관리를 하나의 플랫폼에서 제공합니다. AI 회의 어시스턴트가 회의에 참여하여 회의록을 자동 생성하고, 작업을 자동 등록합니다.

Motivation & Contribution

왜 이 프로젝트가 필요한가, 우리의 기여는 무엇인가

Motivation

🔀 도구 파편화 문제

하나의 프로젝트를 위해 Jira, Confluence, Slack, Zoom을 오가며 컨텍스트 스위칭 비용이 발생

📝 회의 → 작업 단절

회의에서 결정된 내용이 수동 회의록 → 수동 티켓 등록을 거치며 누락/지연 발생

🤖 AI 활용 부재

기존 협업 도구는 프로젝트 맥락을 이해하지 못하는 단순 검색만 제공

Contribution

1. 올인원 협업 플랫폼

문서 편집 + 채팅 + 음성 회의 + 작업 관리를 단일 웹 애플리케이션에서 구현

2. AI 회의 파이프라인

STT → RAG 기반 회의록 자동 생성 → 액션아이템 추출 → 작업보드 일괄 등록

3. RAG 컨텍스트 AI

프로젝트 전체 문서/코드/회의록을 벡터 임베딩하여 맥락을 이해하는 AI 검색
